================================================================
  5. ÉTAPES D'INSTALLATION DU PROJET
================================================================

ÉTAPE 1 — Copier le projet
  Copier le dossier "salonify" depuis la clé USB vers :
  Ex : D:\salonify\  ou  E:\projets\salonify\

ÉTAPE 2 — Ouvrir le terminal dans le dossier du projet
  VS Code → Terminal → Nouveau terminal
  (ou CMD → cd D:\salonify)

ÉTAPE 3 — Installer les dépendances PHP
  > composer install

ÉTAPE 4 — Créer le fichier de configuration
  > copy .env.example .env

ÉTAPE 5 — Générer la clé de l'application
  > php artisan key:generate

ÉTAPE 6 — Créer la base de données MySQL
  → Ouvrir http://localhost/phpmyadmin
  → Nouvelle base de données
  → Nom : salonify
  → Interclassement : utf8mb4_unicode_ci
  → Créer

ÉTAPE 7 — Lancer les migrations et seeders
  > php artisan migrate --seed

ÉTAPE 8 — Créer le lien storage
  > php artisan storage:link

ÉTAPE 9 — Lancer le serveur
  > php artisan serve --port=8000

ÉTAPE 10 — Ouvrir dans le navigateur
  http://localhost:8000

================================================================
  6. CONFIGURATION DU FICHIER .ENV
================================================================

Modifier ces lignes dans le fichier .env créé à l'étape 4 :

--- BASE DE DONNÉES ---
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=salonify
DB_USERNAME=root
DB_PASSWORD=              ← laisser vide si XAMPP sans mdp

--- EMAIL (Mailtrap) ---
MAIL_MAILER=smtp
MAIL_HOST=sandbox.smtp.mailtrap.io
MAIL_PORT=2525
MAIL_USERNAME=VOTRE_USERNAME_MAILTRAP
MAIL_PASSWORD=VOTRE_PASSWORD_MAILTRAP
MAIL_ENCRYPTION=tls
MAIL_FROM_ADDRESS="noreply@salonify.ma"
MAIL_FROM_NAME="Salonify"

--- URL DU PROJET ---
APP_URL=http://localhost:8000

================================================================
  7. AJOUTER PHP AU PATH WINDOWS (si commande php introuvable)
================================================================

1. Rechercher "Variables d'environnement" dans Windows
2. Variables système → Path → Modifier → Nouveau
3. Ajouter : C:\xampp\php
4. OK → redémarrer le terminal
5. Vérifier : php -v

================================================================
  9. PROBLÈMES FRÉQUENTS
================================================================

ERREUR : "php n'est pas reconnu"
  → PHP non ajouté au PATH (voir section 7)

ERREUR : "SQLSTATE Connection refused"
  → MySQL non démarré → ouvrir XAMPP → démarrer MySQL

ERREUR : "Class not found"
  → Relancer : composer install  puis  php artisan config:clear

ERREUR : "The stream or file storage/logs could not be opened"
  → Dossier storage non accessible
  → Windows : clic droit dossier storage → Propriétés →
    Sécurité → Modifier → Tout le monde → Contrôle total
